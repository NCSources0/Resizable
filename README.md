# Resizable
###### A datapack for Minecraft 26.2
Resizable adjusts most of your attributes to match your scale.

It *should* work on any version supporting the attribute command and function macros, but I don't wanna test it.
# Setup
1. Download the first file in the [latest release](https://github.com/NCSources0/Resizable/releases/latest)
2. Copy that file
3. Navigate to `.minecraft/saves` on Windows, it's located at `%appdata%/.minecraft/saves`
4. Find the world folder you want this to be installed on
5. Paste the file in the `datapacks` folder
6. Enter the world in Minecraft
7. Run `/reload`
8. Run `/function ncs:resize {t:"@s", s:\<any real number>}`
9. Have fun!
